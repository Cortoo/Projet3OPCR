Création d'un portfolio d'une architecte d'intérieur dynamique en JavaScript

Lien vers le repo Github : https://github.com/Cortoo/Projet3OPCR

En partant d'une maquette figma existante, je devais dans ce projet développer et intégrer en JavaScript la page de présentation des travaux de l'achitecte, la page de connexion de l'administrateur du site, ainsi qu'une modale (et sa gestion) permettant d'uploader de nouveaux projets et médias. Ce projet m'a permis de comprendre le JavaScript et de l'apréhender. Le backend étant deja développer, je devais y faire appel afin de pouvoir enregistrer les travaux que l'administrateur voulait uploader.
